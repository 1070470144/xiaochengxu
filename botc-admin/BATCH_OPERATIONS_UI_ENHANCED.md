## 💡 设计亮点

### 1. 统一的紫色主题
- 批量操作栏、复选框选中状态都使用紫色渐变
- 形成视觉统一性
- 突出批量操作功能区

### 2. 语义化颜色
- **绿色渐变**：批量通过（正面操作）
- **粉红渐变**：批量删除（危险操作）
- **白色**：取消（中性操作）

### 3. 精致的细节
- 圆角设计（3-6px）
- 阴影效果（立体感）
- hover 光晕（交互提示）
- 滑入动画（引导注意）

### 4. 响应式交互
- hover：上移 + 阴影加深
- active：下压效果
- 过渡：0.3s 平滑动画

### 5. 视觉层次
- 复选框：16px（小巧精致）
- 按钮：中等大小
- 操作栏：突出显示（渐变 + 阴影）

## 🔧 技术实现

### HTML 结构
```vue
<!-- 自定义复选框 -->
<view class="checkbox-wrapper" @click="toggleSelect(id)">
  <view class="custom-checkbox" :class="{ 'checked': isSelected }">
    <text v-if="isSelected" class="checkbox-icon">✓</text>
  </view>
</view>

<!-- 批量操作按钮 -->
<button class="batch-btn batch-approve" @click="batchApprove">
  <text class="btn-icon">✓</text>
  <text>批量通过</text>
</button>
```

### CSS 关键技术
```css
/* 渐变背景 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 阴影效果 */
box-shadow: 0 2px 8px rgba(102, 126, 234, 0.3);

/* 过渡动画 */
transition: all 0.3s ease;

/* hover 上移 */
transform: translateY(-2px);

/* 滑入动画 */
@keyframes slideIn {
  from { opacity: 0; transform: translateY(-10px); }
  to { opacity: 1; transform: translateY(0); }
}
```
