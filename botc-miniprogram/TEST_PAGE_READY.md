## 🐛 如果遇到问题

### 检查清单：
1. ✅ 云对象 `user` 是否已上传？
2. ✅ 本地调试服务是否已禁用？
3. ✅ 网络是否正常？
4. ✅ 手机号格式是否正确？
5. ✅ 是否有其他用户ID可供测试？

### 查看日志：
- 打开浏览器 DevTools Console
- 查找 `✅` 或 `❌` 开头的日志
- 查看详细错误信息

### 常见错误：
- "this.xxx is not a function" → 云对象未上传或未更新
- "无法连接uniCloud本地调试服务" → 禁用本地调试
- "TOKEN_INVALID" → 重新登录
- "用户不存在" → 检查用户ID是否正确
